# Landing Page - Automação RPA com Python

Landing page moderna, persuasiva e responsiva para divulgar serviços de automação de processos com Python, RPA, Selenium, Pandas e integrações com Excel, e-mails e sistemas web.

## Objetivo

Converter visitantes em leads qualificados interessados em:

- automação de tarefas repetitivas
- redução de erros operacionais
- aumento de produtividade
- economia de tempo e custo

## Estrutura

- `index.html` - conteúdo principal da landing page
- `styles.css` - estilos visuais, layout responsivo e identidade tech

## Como usar

1. Abra a pasta `landing-page-rpa-python`.
2. Abra o arquivo `index.html` no navegador.
3. Edite o texto, CTA e links de contato conforme sua necessidade.

## Observação

O projeto foi construído como uma página estática, sem dependências de JavaScript. Isso facilita edição, deploy e manutenção.
